# Birthday Surprise Page

Open index.html in a browser to view the birthday surprise.

To publish it online for sharing:
1. Create a GitHub account.
2. Upload this folder to a new GitHub repository.
3. Go to Settings > Pages.
4. Choose the main branch and save.
5. GitHub will give you a public link like:
   https://your-username.github.io/your-repo-name/

Then send that link to your boyfriend.
